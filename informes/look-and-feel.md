Indicaciones para el Diseño de Interfaz (Look and Feel)

Estilo Visual:

Estilo: "Dark Mode" moderno y minimalista con estética de neón suave.

Contenedores: Usar tarjetas y paneles con esquinas redondeadas y bordes suaves.

Efectos: Emplear sombras sutiles y desenfoques de fondo (glassmorphism) para dar profundidad y sofisticación.

Tipografía: Usar una fuente sans-serif geométrica, clara y moderna, con diferentes pesos para jerarquizar el contenido (ej. Helvetica Neue, Poppins, Inter).

Esquema de Colores (Códigos de Color):

Fondo Principal (Dark): #111317 (Un negro azulado muy oscuro para el fondo general del panel).

Paneles Secundarios: #1C1F26 (Un gris azulado muy oscuro para diferenciar las tarjetas y secciones).

Texto Principal: #FFFFFF (Blanco puro para el texto de mayor importancia).

Texto Secundario/Desactivado: #A1A7B3 (Un gris claro azulado para texto menos relevante).

Color de Acento Principal (Verde Neón):

Fondo Degradado de la Tarjeta Principal (ej. "Total balance"): #A6F684 a #FFFFFF (Un degradado de verde lima brillante a blanco).

Indicadores Positivos (ej. +15.7%): #A6F684 (Verde lima brillante).

Color de Acento Secundario (Púrpura Neón):

Degradado de Tarjetas de Crédito/Gráficos (ej. "My cards", "Revenue flow"): #BB95FD a #985EFD (Un degradado de púrpura claro a púrpura vibrante).

Detalles en Gráficos (ej. un segmento del gráfico circular): #BB95FD.

Color de Alerta/Gasto (Rojo/Naranja Suave): #FF6B6B (Para números negativos, ej. -1,124$ o -10.7%).

Color de Fondo de Botones/Etiquetas: #000000 o #1C1F26 con texto #FFFFFF.

Color de Borde Sutil: #2D323C (Para bordes de inputs, divisores finos).

Composición y Estructura:

Diseño: Utilizar un diseño de rejilla (grid layout) flexible para organizar los paneles.

Encabezado: Mantener un encabezado de panel oscuro con el logo a la izquierda, menú de navegación centrado (con un "píldora" de fondo para el elemento activo) y elementos de perfil/búsqueda a la derecha.

Organización del Contenido: Agrupar la información en tarjetas modulares con títulos claros.

Jerarquía: Usar el tamaño de fuente y el peso para destacar los números y títulos principales (ej. el balance principal es mucho más grande que el texto secundario).

Espaciado: Asegurar un espaciado generoso entre elementos y paneles para un diseño limpio y respirable.

Uso de Elementos Visuales:

Gráficos: Utilizar gráficos de barras y circulares con los colores de acento (verde, púrpura, etc.) para visualizar datos.

Iconos: Usar iconos de línea finos y modernos para representar acciones y categorías.

Tarjetas: Usar tarjetas con degradados de color vibrantes para elementos clave como "Mis tarjetas", asegurando que los degradados sean suaves y no saturados.

Visualización de Transacciones/Suscripciones: Listas limpias con iconos de aplicaciones y logotipos, montos y detalles relevantes, usando los colores de acento para montos negativos o positivos si aplica.

Instrucciones para Aplicar a Nuevo Contenido:

Adaptación: Sustituye el contenido financiero por el nuevo contenido (ej. datos de un proyecto, métricas de marketing, información de usuarios, etc.).

Mantenimiento: Sigue estrictamente el esquema de colores, la tipografía y el uso de sombras/desenfoques para asegurar que la nueva página mantenga el mismo "look and feel".

Creatividad: Utiliza los colores de acento para destacar la información más importante del nuevo contenido de manera similar a como se hace en el diseño original.
